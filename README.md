# Superstore Sales Analytics

## Project Overview

This project analyzes sales, profit, customer, and product performance using SQL and Power BI. The objective is to transform raw transactional data into actionable business insights that support data-driven decision-making.

---

## Business Problem

Organizations need visibility into sales trends, profitability drivers, customer behavior, and product performance in order to make informed business decisions.

This project explores a retail sales dataset to identify:

* Top-performing products
* Most profitable customer segments
* Regional sales performance
* Revenue trends
* Profitability opportunities
* Business growth insights
